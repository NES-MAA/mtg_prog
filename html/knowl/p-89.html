I used binary logistic regression to construct a model to answer the question: which variables among region, urbanicity, number of parents in a household, parental education level, religiosity, marijuana use, and gender predict adolescent alcohol use? I used the 2015 12th Grade Monitoring the Future dataset to construct a model which predicts the likelihood of an adolescent using alcohol. I concluded that the model including gender, marijuana use, urbanicity, mother education level, and religiosity significantly predicts adolescent alcohol use. (Advisor: Peter Johnson)
