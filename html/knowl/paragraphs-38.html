Well-dominated Graphs

Kirsti Kuenzel   —   Western New England University

Given a graph \(G\text{,}\) a set \(S\subseteq V(G)\) is a dominating set of \(G\) if every vertex in \(G\) is either in \(S\) or adjacent to a vertex in \(S\text{.}\) We say that \(G\) is well-dominated if all minimal dominating sets have the same cardinality. In 1988, Finbow, Hartnell, and Nowakowski classified all well-dominated graphs of girth at least \(5\text{.}\) In this talk, we discuss the relationship between the class of all well-dominated graphs and the class of all well-covered graphs. We also discuss open problems in this area.
