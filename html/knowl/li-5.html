Random Matrix Theory and the Riemann Zeta Function

The talk will cover some introductory concepts in complex analysis in order to discuss the Riemann Zeta Function. Some properties and forms of the function will be demonstrated, so that the Riemann Hypothesis can be discussed. The primary purpose of the project is understanding the Hilbert-Polya conjecture and the possible connection between the distribution of zeros of the RZF and the Gaussian Unitary Ensemble.
